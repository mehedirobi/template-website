# Dark UI Template Website

A modern Dark UI landing page template designed for Webflow-style agencies and creative businesses. The project focuses on minimal layout, visual hierarchy, and branding-ready components.

## Features

- Clean Dark UI aesthetic
- Responsive Header + Hero section
- Call-to-Action (CTA) for Live Preview
- Productive user stats (50K / 2M+ / 95%)
- Testimonial section with 5-star ratings
- Contact form layout with Name, Email, Message fields
- Structured Footer with navigation links
- Uses Google Fonts + Font Awesome

## Tech Stack

- HTML5
- CSS3
- Font Awesome Icons
- Google Fonts (Raleway)
